# Get started
## at directory 'cmu-score-announcement'
## 1. run frontend
###   1.1) open first terminal
###   1.2) run ``` cd frontend ```
###   1.3) run ``` npm start ``` 
## 2. run backend
###   2.1) open second terminal
###   2.2) run ``` cd backend ```
###   2.3) run ``` python server.py ```
